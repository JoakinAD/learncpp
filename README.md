## Sobre este repositorio

Repositorio usado para contener y revisar mis avances en el recurso de aprendizaje [learncpp.com](https://www.learncpp.com/), a través del cual planeo aprender C++ desde 0 hasta alcanzar un nivel considerable.

> Al ser un proyecto completamente dedicado al aprendizaje, la práctica pura de la programación y la resolución de problemas, **no se han usado en ningún momento herramientas de IA como Claude, ChatGPT o Gemini,** ni para la escritura, ni para la completación, ni siquiera como apoyo a la hora de hacer código o resolver problemas.
